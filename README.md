# Recipe ratings analysis
Author: Daniel Kong
Final project for my DSC 80 class

## Introduction
This project explores the relationships between recipe attributes (e.g., cooking time, protein content, and tags) and their ratings on food.com. The goal is to understand how these factors influence ratings and predict a recipe's average rating using key features.

We use the **Recipes and Interactions** dataset from food.com, which contains:
- Over 230,000 recipes (rows in the dataset) with metadata (columns) like cooking time, number of ingredients, and nutrition information.
- Ratings from users who interacted with these recipes.

Through this analysis, we aim to:
1. Identify trends in recipe ratings.
2. Build predictive models for average ratings.
3. Assess fairness in our predictions.****
